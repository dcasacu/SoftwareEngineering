# Skills

Skills are installed via `npx autoskills` and provide specialized instructions for specific tasks.

## Installed Skills

### frontend-design

Create distinctive, production-grade frontend interfaces with high design quality.

- **Source**: anthropics/skills
- **Use when**: Building web components, pages, React components, HTML/CSS layouts, or styling/beautifying web UI.

### accessibility

Audit and improve web accessibility following WCAG 2.2 guidelines.

- **Source**: addyosmani/web-quality-skills
- **Use when**: "improve accessibility", "a11y audit", "WCAG compliance", "screen reader support", "keyboard navigation", or "make accessible".

### seo

Optimize for search engine visibility and ranking.

- **Source**: addyosmani/web-quality-skills
- **Use when**: "improve SEO", "optimize for search", "fix meta tags", "add structured data", "sitemap optimization", or "search engine optimization".

## Installation

Skills are managed via the `skills-lock.json` file. To install or update skills, run:

```bash
npx autoskills
```

# LineUp - Software Engineering Project

LineUp is a mobile application designed to improve the queue experience in local markets. The project focuses on helping customers check queues, join them, follow their position and receive feedback.

The application also includes shop-owner features, allowing market stalls to manage their own queues in a simple way.

This repository was created as part of the Software Engineering course and follows an agile development process using GitHub Projects / Kanban.

## Project Goal

The main goal of LineUp is to reduce uncertainty and waiting time in local markets by providing a digital queue management system.

The app allows users to:

- Continue without logging in as a guest.
- Log in when needed.
- View market and shop information.
- Join and follow queues.
- Receive feedback when their turn is close.
- Support accessibility features such as vibration notifications.
